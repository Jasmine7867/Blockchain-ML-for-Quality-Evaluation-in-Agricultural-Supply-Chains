#####  Blockchain-ML-for-Quality-Evaluation-in-Agricultural-Supply-Chains
This project integrates Blockchain and Machine Learning to enhance transparency, traceability, and quality evaluation in agricultural supply chains. It analyzes crop quality data using ML models and securely records verified results on blockchain. The system reduces fraud, builds stakeholder trust, and improves food safety from farm to consumer.


###### 📌 Project Description
This project integrates Blockchain Technology and Machine Learning (ML) to enhance transparency, traceability, and quality evaluation in agricultural supply chains. The system securely records supply chain data on blockchain and uses ML models to analyze and predict product quality, helping stakeholders ensure food safety and reduce fraud

###### 🎯 Objectives
1. Improve transparency and traceability in agricultural supply chains.
2. Securely store transaction and quality data using blockchain.
3. Predict crop quality using machine learning models
4. Reduce fraud, contamination, and inefficiencies.
5.Enable real-time monitoring from farm to consumer
